1. Passado Presente e Futuro do .NET

Microsoft iniciou nos anos 70 criando linguagens de programação: Basic


Nos próximos anos a microsoft atua fortemente na criação do OS windows


No final dos anos 90, Microsoft tentou consolidar as ferramentas de devs (IDES e runtimes) com o VS 97

1997

+ VB 5
+ VFoxPro 5
+ C ++ 5
+ J++ 



1998

+ VB 6
+ VFoxPro 6
+ C ++ 6
+ J++ 6


1999

Scott Guthrie criou uma ferramenta web com java, e a chamou de ASP + (depois chamou ASP Next e depois ASPX)

Jason Zander ajudou na criação de um common runtime para VB e C++ (CLR)


Java estava bem! Então a Sun Microsystems fez um acordo para a microsoft não mezer mais com java.

Com isso Anders Hejsberg começou a trabalhar com C#

2000

Microsoft lança o novo ambiente de desenv .NET 1.0 - inicialmente chamado de Next Generation windows Services (NGWS)


2001

Miguel de icaza começa a trabalhar no projeto Mono, uma reimplementação black box do .NET, open source e multiplataforma


2002

Lançamento do VS .NET com C# 1.0

Conhecido como 22 linguagens, 1 plataforma:

.C#.NET
.C++.NET
.VB.NET
.J#.NET
.others

2003

Lançamento do .NET 1.1 com o VS 2003

Trabalham em melhorias na CLR para lançar a CLR2



2005


Lançamento do .NET 2.0 com C# 2.0 no VS 2005

Microsoft começa a atingir seu objetivo inicial, inclusive evoluindo na web.



2007 - 2008

Lançamento do .NET 3.5 com C# 3.0 no VS 2008, com Silverlight, WPF e WCF

Microsoft contrata um time de pessoas que tinham uma pegada open source e começam a atuar na criação do ASP.NET MVC

Começa a se falar em Windows Azure


2010

Lançamento do .NET 4.0 com C# 4.0 no VS 2010, também com F#

Microsoft lança comercialmente windows Azure

Anders Hejlsberg começa a trabalhar no typescript



2011

Miguel de Icaza inicia Xamarin, basicamente ele desenvolvia em C# aplicativos que rodam em Android e IOs


2012

Lançamento do .NET 4.5 com C# 5.0 no VS 2012

Lançamento do Typescript


2013

Lançamento do .NET 4.5.1 no VS 2013

Inicio do Roslyn, um novo compilador para C# e VB.NET

Microsoft continua atuando mais na frente JS e aumenta também a incorporação de ferramentas open source ao ambiente

Já temos aqui o ASP.NET mais consolidado com MVC, web API e SignaIR

Mas, tudo ainda é muito windows


2014


satya Nadella se torna CEO da Microsoft e direciona o foco da empresa para cloud

Criação do .NET Foundation para gestão de projetos em open source


2014

Windows azure passa a se chamar Microsoft Azure


É intrduzido o conteito do ASP.NET vNext, posteriormente chamado de ASP.NET Core


2015

Lançamento do .NET 4.6 com C# 6.0 no VS 2015


Lançamento do VS Code


2016

Microsoft adquire a Xamarin e adiciona o produto como parte de sua stack .NET e projetos open source. 

Lançamento do VS for Mac

Lançamento do .NET Core 1.0 - Totalmente novo, open source e multiplataforma



2017

Lançamento do .NET Framework 4.7 com C# 7.0 no VS 2017

Lançamento do .NET Core 2.0 com C# 7.0 no VS 2017, VSCode ou VS for Mac 2017



Lançamento do .NET Core 4.8 com C# 7.3 no VS 2019

Lançamento do .NET Core 3.0 com C# 8.0 no VS 2019, VSCode ou VS for Mac 2019


2020

.NET Framework está pronto na versão 4.8 e deixa de ser evoluido - junto com ele WCF e ASP.NET Webforms


Previsão lançamento do .NET 5
